# My First Startup - Countdown (Vite + React)

- Background: `#fafafa`
- Text: `#000000`
- Shows days remaining
- Small bottom timer: `HH:MM:SS`
- Title at top: "My First Startup"
- Bottom shows total GitHub commits for `pragnesh64/my-first-startup`

## Run locally

```bash
npm install
npm run dev
```

Then open the printed local URL.

## Configure launch date

Edit `src/App.jsx` and change `TARGET_DATE_ISO` to your desired ISO date/time.
